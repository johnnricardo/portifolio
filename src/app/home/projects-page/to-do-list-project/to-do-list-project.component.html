TO-DO LIST

(link do projeto)

Esse foi o meu primeiro projeto Angular! Nesse projeto entendi como funciona comunicação entre componentes com @input e @output, além de entender como funcionam os componentes e modulos nesse framework!

Funcionalidades:

- Quando o usuário digita uma tarefa e presiona enter, o valor do input é passado para um método no componente de lista que preenche a ultima posição do Array de listagem com o valor do input

- Quando o usuário clica na lixeira, o id daquele objeto é passado para um método no componente que remove ele do Array

- Quando o checkbox é marcado, é chamado uma função que leva o objeto para o final da lista e taxa o texto contido neste objeto, através do seu id, passado na chamada do método

- Quando existe alguma tarefa, o ‘botao ‘’DELETAR TODAS AS TAREFAS’’ é mostrado(reativo) e quando esse botão é clicado uma função é chamada para atribuir ao Array de listagem o valor de vazio ( [ ] )
